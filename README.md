# FUDLEVI ($FUDLEVI) Website

The bull that carries your bags. No FUD. Only LEVI. 🐂💰

## Run locally

```
npm start
```

Then open http://localhost:3000

## Deploy on Railway

Railway auto-detects the `start` script — no build step, no dependencies.

## Updating at launch

Search `index.html` for these placeholders and replace them:

- `COMING SOON — LAUNCHING ON PUMP.FUN` → your contract address
- `https://pump.fun` → your pump.fun coin page URL
- `https://x.com` → your X/Twitter profile
- `https://t.me` → your Telegram group
- `https://dexscreener.com` → your Dexscreener pair URL
